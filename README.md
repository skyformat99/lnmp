# 欢迎使用 LNMP一键部署脚本

------

**LNMP一键部署脚本**是一键部署web环境所用，非常适合网站运维人员，开发人员在Linux系统中部署web环境。 您可以使用LNMP一键部署脚本：

> * 自动部署Nginx,Mysql,Php
> * 备份数据库
> * 升级软件版本
> * 添加虚拟站点和目录
